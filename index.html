Haqqımızda

ByteLab olaraq məqsədimiz gəncləri praktiki dərslər, real layihələr və praktiki təcrübələr vasitəsilə proqramlaşdırma dünyasına yönləndirməkdir. Proqramlaşdırma öyrənməyin ən yaxşı yolu sınaqdan keçirmək və bu prosesdən zövq almaqdır. Kursumuz backend inkişafı (data struktur/alqoritm), verilənlər bazası və Java, C# kimi müasir texnologiyalarda güclü təməl qurmaq üçün xüsusi olaraq hazırlanıb. Şagirdlər gələcəkdə ya mühəndis olmağı hədəfləyə və ya sadəcə texnologiyanın necə işlədiyini öyrənə bilərlər — ByteLab onlara texnologiya yolculuğunda ilk inamlı addımlarını atmağa kömək edəcək.

Kursa yazılmadan əvvəl nümunə dərslərə baxa və ya bizə yazaraq tədris planını nəzərdən keçirə bilərsiniz.

Niyə Kodlaşdırma Öyrənilməli?

Kodlaşdırma sadəcə ekranda sətirlər yazmaq deyil — bu həm də gələcəyin başlangıcıdır. İstifadə etdiyin hər bir tətbiq, oyun və vebsayt, kodlaşdırma bacarığı olan biri tərəfindən yaradılıb. Bu, ideyalarınızı reallığa çevirən bir bacarıqdır. Kodlaşdırma öyrəndikcə, fərqli düşünməyi öyrənirsən. Sadəcə təlimatları izləmir —həm də problemləri həll edir, heç nədən bir şey yaradır və texnologiyanı öz xeyrinə işlədə bilirsən. Kodlaşdırma sənə ətrafındakı dünyanı formalaşdırmaq gücü verir , öz vebsaytını yaratmaqdan tutmuş insanlara fayda verən ağıllı sistemlər qurmağa qədər.

Kurs idarəetmə sistemi

Bu layihə tam funksional Kurs İdarəetmə Sistemidir. Sistem tələbələrin qəbulu, dərs cədvəllərinin və imtahanların idarə olunması, nəticələrin analizi, həmçinin ödəniş və mühasibatlıq əməliyyatlarının aparılması üçün hazırlanmışdır. Backend tərəfi Java Spring Boot framework-ü üzərində qurulub və layihədə Layered Architecture Pattern tətbiq edilib. İstifadəçi identifikasiyası və təhlükəsizliyi JWT əsaslı authentication vasitəsilə həyata keçirilir.Məlumatların saxlanması üçün PostgreSQL istifadə olunub və ORM qatında MyBatis inteqrasiya edilib. Sistem frontend tətbiqindən daxil olan sorğuların emalı üçün optimallaşdırılmış REST API-lər üzərində işləyir. Bundan əlavə, günün sonunda avtomatik hesabatların göndərilməsi üçün schedule mexanizmləri qurulmuşdur. Bu, sistemin operativ və davamlı şəkildə işləməsinə, həmçinin idarəçilərə gündəlik məlumat axını təmin etməsinə imkan verir.

Bu layihə ISO standartlarına əsaslanan hesabatların dəstəklənməsi və yaradılması üçün hazırlanmış iki hissəli tətbiqdir. Sistem istifadəçilərə məlumatları səmərəli şəkildə daxil etməyə, idarə etməyə və qiymətləndirməyə imkan verir və eyni zamanda ISO tələblərinə tam uyğunluğu təmin edir. Sistem HttpContext və təhlükəsiz API inteqrasiyası vasitəsilə problemsiz ünsiyyət quran iki bir-biri ilə bağlı tətbiqdən ibarətdir və modullar arasında dəqiq və sinxron əməliyyatları təmin edir. Üstünlükləri: - Gündəlik tapşırıqların və fəaliyyətlərin rəqəmsal həlli - Tapşırıqların icra vəziyyətinin real vaxt izlənməsi - Sübut sənədlərinin əlavə olunma imkanı - Məsul şəxslərə avtomatik e-poçt bildirişləri - Təqdim edilmiş tapşırıqlar üçün icra hesabatları - Hər tapşırıq avtomatik nömrələnir və qeydiyyata alınır.

Onlayn Əczaçılıq

Bu layihə onlayn apteklər üçün stok idarəetmə sistemidir. Modul əsaslı backend həlli sayəsində sistem genişlənə bilir və asanlıqla idarə oluna bilər, müəssisə səviyyəli tətbiqlər üçün uyğundur. Layered Domain-Driven Design (DDD) və təmiz arxitektura prinsiplərinə əsaslanaraq, məlumat əldə etmə, biznes məntiqi, paylaşılan modellər və lokalizasiya resursları üzrə məsuliyyətləri aydın şəkildə ayırır.Sistem inventar, istifadəçilər və inzibati əməliyyatları idarə etmək üçün güclü və təhlükəsiz RESTful API-lər təqdim edir. İnteqrasiya olunmuş lokalizasiya dəstəyi və Docker ilə yerləşdirmə müxtəlif mühitlərdə elastikliyi, asan baxımı və ardıcıllığı təmin edir. Bu isə layihəni müasir DevOps iş axınları üçün ideal edir.
